以太坊私有链搭建踩坑

安装 Geth

MacOS 安装 Geth

mac先装homebrew的包管理器
https://brew.sh/index_zh-cn

安装 Geth
$ brew tap ethereum/ethereum
$ brew install ethereum

Ubuntu 安装 Geth

$ sudo add-apt-repository -y ppa:ethereum/ethereum
$ sudo apt-get update
$ sudo apt-get install ethereum

安装 Mist钱包

https://github.com/ethereum/mist/releases

找到对应系统 Ethereum-Wallet 安装包.
下载完不要同步区块, 只是用来做 私链 前端界面.

创建创世区块文件

MacOS 打开终端 cd 到 ~/Library/Ethereum/目录下.

Linux 打开终端 cd 到 ~/.ethereum 目录下.

删除目录下的 文件:
$ rm -rf *

创建 创世区块文件:

$ cat > genesis.json
输入:

control + C 结束输入

初始化区块链

$ geth init genesis.json

启动私有链节点

注意: 如果提示 30303端口被占用,解决方法:

查看占用进程PID
$ lsof -i:30303

kill 占用进程
$ kill -9 进程号

然后, 重新执行启动节点命令

私有链 挖矿

新开 一个终端, 打开 Geth控制台.

$ geth attach 'http://127.0.0.1:8545'

在控制台中, 新建一个以太坊 钱包账户:

> personal.newAccount('新钱包密码')

开启 挖矿 !

> miner.start(1)

可以看到Geth节点终端下, 已经开始挖矿

打开 Mist 钱包 也能查看 Ether

停止挖矿

> miner.stop()
