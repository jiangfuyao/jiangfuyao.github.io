Ubuntu 区块链开发go语言，Node.js，truffle环境搭建

区块链开发一般在Linux和 Mac OS下进行。
虚拟机安装或者双系统的教程 谷哥度娘都有很多详细的教程，不锦上添花。
Go语言很适合开发区块链后端，砍柴需要需要先磨刀，学习开发区块链从学习Go开始是一个很好的选择。

So… Go起！

Golang install:

https://www.golangtc.com/download

根据操作系统下载对应的软件包

解压

$ tar -C /usr/local -zxf go1.3.3.linux-amd64.tar.gz

Golang environment(Go语言环境)

设置环境变量
（1）打开配置文件
运行以下命令:
$ gedit ~/.bashrc

（2）添加命令

在文件尾部添加以下命令：
export PATH=$PATH:/usr/local/go/bin
export GOPATH=/home/gopath

（3）命令生效
$ source ~/.bashrc

测试Go语言环境

在终端中运行Go命令：
$ go或$ go version

Golang IDE:

根据自己的喜好选择IDE，推荐Atom，酸爽，插件很赞。

Sublime

Sublime官网

搜索并安装goSublime插件
crtl+b可以调出命令行，来编译执行go代码

LiteIDE

下载地址

下载的文件是liteidex30.2.linux64-qt4.tar.bz2（不同操作系统版本其文件名可能不同）。

解压缩安装

将LiteIDE软件安装到目录/opt下。
需要使用以下命令，以root权限打开文件管理器：
$ sudo nautilus

将下载的文件liteidex30.2.linux64-qt4.tar.bz2拷贝到目录/opt下，然后解压缩（提取）该文件。此时会生成目录/opt/liteide。LiteIDE软件在该目录中。
上述操作完毕后，将文件liteidex30.2.linux64-qt4.tar.bz2删除。

运行LiteIDE

点击/opt/liteide/bin/liteide即可运行LiteIDE软件。

推荐一个不错的暗色主题 : )

Atom

官网下载地址
下载安装Atom。

Atom实用插件

go-plus Go语言插件
autocomplete-solidity代码自动补齐
linter-solium、linter-solidity代码错误检查

Chinese(Simplified) Menu 两个汉化插件
file-icons 文件图标
