Hello World

每门语言的Hello Wrold,都是一块敲门的板砖。

在Ubuntu终端下cd 到项目路径，
运行atom ./用atom编辑器打开当前目录
在Atom中新建一个helloworld.go文件，编辑完之后保存，
在Ubuntu终端下，运行:
$ go run helloworld.go
看到Hello,world 输出，则成功。

变量的声明

变量的声明有好多种方法：
格式1： var name type

例如：
var number int
number = 500

格式2: name := value //自行判断变量类型，一旦判断不可改变

例：a := 100

多变量的声明：

格式1: var name1, name2, name3 type

例如： var a, b, c int

格式2:
var {
name1 type
name2 type
}

例如：
var {
a int
b string
c float64
}

string 字符串类型的变量声明：
var a string = “abc”

常量声明

const b string = “abc”
const b = “abc”
const L int = 10 //显示类型
const M = 5 //隐藏类型
常量 变量名一般为 大写 const LENGTH int = 10
